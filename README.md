**Flor de Lis - Um Site para Leitura Digital**

O projeto foi voltado para a finalização do curso de Front-End. Desenvolvido no intuito de facilitar a leitura dos usuários.

*Equipamentos Utilizados*
- HTML / CSS / JavaScript;

*Funcionalidades Implementadas*

- Filtro de busca em tempo real;
- Filtros por status;
- Responsividade para plataformas;
- Pesistência com LocalStorage.

*Como Vizualizar*

1. Clone este repositório.
2. Abra o arquivo `index.html` no seu navegador.
